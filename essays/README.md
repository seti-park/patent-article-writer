# essays/ — tracked deliverables

One directory per essay edition; the root holds ONLY current, publishable deliverables.

baselines, retro material) in the same commit that lands their replacement, with a
SUPERSEDED banner on their README. Agents and humans: grounding sources and publication
material come from the root dirs only; never from `_superseded/`.
