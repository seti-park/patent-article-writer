## 2026-07-12 — tesla-us20260196678-hemmed-tabless (교정 5쌍)

1. **수사는 단위명사형으로**
   - ✗ "지금까지 길은 둘이었습니다."
   - ✓ "지금까지 길은 두 가지였습니다."
   - ✗ "셋째 길을 청구합니다"
   - ✓ "세 번째 길을 청구합니다"

2. **화제는 평서문으로 연다 — 수사 의문문·문답 에코 회피**
   - ✗ "접기만 하면 무엇이 달라질까요? 먼저 공장이 달라집니다."
   - ✓ "접기의 효과는 공장에서 먼저 나타납니다."
   - (중간 시도 "왜 중요할까요? 먼저 공장이 달라집니다"도 탈락 — 문답 장치
     자체를 버리고 평서형 화제 문장으로 여는 쪽이 이겼다.)

3. **문서는 행위자가 아니라 장소 — 그리고 열거가 이미 증거면 서술하지 않는다**
   - ✗ "명세서가 장비 이름까지 짚어 적습니다."
   - △ "명세서에는 이 장비들의 이름이 하나하나 적혀 있습니다." (구문은 교정됐으나)
   - ✓ (삭제) — 앞 문장이 다섯 장비를 이미 열거하므로 메타 서술은 군더더기.

4. **전환은 은유가 아니라 실질로 — 조사 하나가 전환을 질 수 있다**
   - ✗ "성능 논리도 같은 그림에서 나옵니다." (추상 복합명사 + 은유 전환문)
   - ✓ (삭제 + "내부 저항을" → "내부 저항**까지**") — '까지' 하나가 공장→성능
     전환을 짊어진다. 스터디팩 원문("내부 저항과 안전성까지")과 결이 같다.

5. **판정 기준: 학생에게 소리 내어 말할 문장인가** — 아니면 재작성.
